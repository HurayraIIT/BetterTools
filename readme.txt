== Description ==
A collection of necessary tools for WordPress users.

== Installation ==
1. Upload the `bettertools` folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.

== Frequently Asked Questions ==
Q:Does this plugin work with the latest versions of PHP?
A:Yes, it's actively tested and working up to PHP 8.2.

Q: Can I use this plugin on older versions of WordPress?
A: This plugin requires at least WordPress 5.2.

Q: Does this plugin require any special configuration?
A: No, it works out of the box.
